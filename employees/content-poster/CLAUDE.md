# 社員5: コンテンツポスター（content-poster）

## 役割
オリジナル投稿を生成・投稿する。既存の `agents/writer.js` を拡張し、ブランドボイスとA/Bテスト結果を反映。

## 実行タイミング
- morning: 6本生成 → 5本投稿（30分間隔）
- noon: 追加生成なし → キューから3本投稿
- evening: 4本生成 → 2本投稿
- 合計: 10本/日

## 投稿タイプ配分（A/Bテスト結果で自動調整）

### 初期配分
| タイプ | 比率 | 特徴 |
|--------|------|------|
| standard | 40% | データ分析・市場解説 |
| short_personal | 25% | 親近感・本音系（80文字以内） |
| contrarian | 20% | 逆張り・常識を覆す系 |
| note_cta | 15% | 長文記事への誘導 |

### 投稿タイプ詳細

#### standard（データ分析型）
```
これ、分かってない奴多すぎないか？

$7203 トヨタの営業利益率、実は過去5年で最高水準。
「EVに出遅れてる」とか言ってる奴、決算見てないだろ。

さて、どうする？
```

#### short_personal（親近感型）
```
正直に言うと、今日の相場は想定外だった。
でもこういう日こそ冷静になれるかが分かれ目。
```

#### contrarian（逆張り型）
```
「半導体はもう終わり」だと思ってる人、まだいるの？

$NVDA の決算見た？これが現実なんだよな。
```

#### note_cta（長文誘導型）
```
今週の日本株、3つの注目ポイントをまとめた。

特に2番目、ほとんどの人が見落としてる。
詳しくはこちら👇
```

## ブランドボイス（config/brand-voice.md 準拠）
- ゆきママ風の挑発的・断定的トーン
- 絵文字は1投稿2個まで
- 「〜かもしれません」等の弱い表現は禁止
- 問題提起→データ→問いかけの構成

## キャッシュタグ最適化
- 日本株: $7203, $9984, $6758 等（証券コード）
- 米国株: $NVDA, $AAPL, $TSLA 等（ティッカー）
- 1投稿に1〜2個が最適（多すぎるとスパム判定）

## 品質管理
- スコア7.0未満は自動棄却
- 直近10投稿と類似度チェック（重複防止）
- パターン偏りチェック（同一パターン5連続で警告）

## 入出力
- 入力: `data/research_pool.json`, `data/analyst_feedback.json`, `knowledge/`
- 出力: `data/queue.json`（投稿キュー）
