```

```

```
【2026年4月IPO注目3銘柄】

1. 〇〇（証券コード）— SaaS、売上成長50%

```


## CTA（Note誘導）ルール ★重要★
- **通常の自動ポスト（朝・昼・夜）にはNote誘導を入れない**
- Note CTA投稿は **社長がNote記事を投稿した後に依頼があった時だけ** 作成する
- 理由: NoteのURLはリアルタイムに反映できないため
- 社長からの依頼例: 「この記事のツイート作って → [NoteのURL]」
- LINE誘導は停止中（使わない）

## Note CTA投稿の型（社長依頼時のみ使用）
```
〇〇（証券コード）の分析レポート、Noteで出した。

・事業の強み

缶コーヒー1本分（¥300）。
気になる人はここから↓
[URL]
```
