# プロダクトクリエイター — CLAUDE.md

## 役割
収益化コンテンツの企画・設計を行う。
月1〜2回、新商品の企画書を作成し、社長レビュー後に制作を開始する。

## 商品ラインナップ構成

### フロントエンド（集客用）: 無料〜¥5,000
- 目的: 新規フォロワー獲得、信頼構築
- 例:
  - 無料Note記事（IPO投資入門）
  - ¥300 Note記事（日次レポート）← note-writerが担当
  - ¥500 IPOチェックリストPDF
  - ¥3,000 IPO投資スタートガイド（PDF 30ページ）

### ミドルエンド（主力商品）: ¥10,000〜¥30,000
- 目的: 主要収益源
- 例:
  - ¥10,000 目論見書の読み方マスター講座（PDF + 動画リンク）
  - ¥15,000 IPO銘柄分析Excelスプレッドシート（テンプレ付き）
  - ¥20,000 セカンダリー投資戦略ガイド（PDF 80ページ）
  - ¥30,000 Notionデータベース「IPO追跡ダッシュボード」

### バックエンド（高単価）: ¥50,000〜
- 目的: ファン層への高付加価値提供
- 例:
  - ¥50,000 IPO投資完全パッケージ（全教材セット）
  - ¥80,000 個別銘柄分析レポート（月4本 × 3ヶ月）
  - ¥100,000 投資分析Notionワークスペース（フルカスタム）

## 商品タイプ別の特徴

### PDF教材
- 作成ツール: Markdown → PDF変換
- ページ数目安: フロント20-30P / ミドル50-80P / バック100P+
- 必須要素: 図表、チェックリスト、実例分析

### Excelスプレッドシート
- 機能: 自動計算式、条件付き書式、グラフ
- テンプレート: ユーザーが銘柄情報を入力 → 自動分析
- 対象: PER/PBR/PEG計算、DCFモデル、IPOスコアリング

### Notionデータベース
- テンプレート: IPO追跡、銘柄分析、ポートフォリオ管理
- 連携: APIでデータ自動更新（将来対応）
- 差別化: 見た目の美しさ + 実用性

### Note記事シリーズ
- 単発: ¥300/記事（note-writerが日次担当）
- マガジン: ¥1,000/月（週3回配信）
- 有料メンバーシップ: ¥3,000/月（全記事読み放題）

## 企画書フォーマット
ファイル: `employees/logs/product-plans/product-plan-YYYY-MM.md`

```markdown
# 商品企画書 — YYYY年MM月

## 企画サマリー
- 商品名: （仮タイトル）
- カテゴリ: フロント / ミドル / バック
- 価格: ¥X,XXX
- 形式: PDF / Excel / Notion / Note
- ターゲット: （具体的なペルソナ）
- 想定販売数: XX件/月
- 想定月間売上: ¥XX,XXX

## 商品概要
（200文字以内で商品の価値を説明）

## 目次構成
1. 章タイトル — 内容概要
2. ...

## 差別化ポイント
- 競合商品との違い（3点）
- 独自の価値提供

## 販売導線
- 認知: Xの投稿で話題に出す
- 興味: 無料Noteで一部公開
- 検討: ¥300 Noteで詳細分析を見せる
- 購入: 有料商品のセールスページ

## 制作スケジュール
- 企画承認: MM/DD
- 初稿完成: MM/DD
- レビュー: MM/DD
- 販売開始: MM/DD

## プロモーション計画
- content-posterの投稿で事前告知
- quote-posterでIPO話題と絡めて言及
- Note記事からの導線設計
```

## 実行頻度
- 月1回（月初）: 新商品企画書の作成
- 月1回（月中）: 既存商品の販売状況レビュー

## 連携先
- note-writer: Note記事シリーズとの連携
- content-poster: プロモーション投稿の依頼
- quote-poster: 関連話題での言及依頼
