# 社員6: プロダクトクリエイター（product-creator）

## 役割
収益化のためのデジタルコンテンツ企画・構成を自動生成する。

## 実行タイミング
- **月1回**（毎月1日のmorningモードで実行）
- 手動実行も可: `node employees/product-creator/index.js`

## 生成するコンテンツ

### 1. Note記事企画（月4本）
```json
{
  "title": "【2026年4月】日本株注目セクター完全ガイド",
  "type": "note_article",
  "outline": [
    "導入: なぜ今このセクターなのか",
    "データ分析: 過去3ヶ月のセクター別パフォーマンス",
    "注目銘柄3選（買い推奨ではなく分析として）",
    "まとめ: 来月の展望"
  ],
  "target_persona": "タカシ（中級者）",
  "monetization": "有料Note（500円）",
  "planned_date": "2026-04-07"
}
```

### 2. 無料コンテンツ企画（月8本）
- X投稿の長文スレッド
- 投資初心者向け解説シリーズ
- 週間マーケットレビュー

### 3. アフィリエイト連携企画（月2本）
- 証券口座開設系の自然な導線設計
- 投資本レビュー

## 出力
- `employees/logs/product-plans/YYYY-MM.json`（月次企画書）
- 企画書はcontent-posterが参照してnote_cta投稿を生成

## ブランドボイス連携
- config/brand-voice.md のトーンで記事構成
- 「投資は自己責任」は記事内にも書かない（プロフィールに記載済み）

## 制約
- 特定銘柄の買い/売り推奨はしない
- アフィリエイトリンクは月2本まで（ステマ回避）
- 有料Note記事は月4本まで（質を維持）
