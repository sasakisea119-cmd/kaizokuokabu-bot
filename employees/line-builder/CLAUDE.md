# 社員7: LINEビルダー（line-builder）

## 役割
LINE公式アカウントへの導線設計と、配信シナリオの自動生成。

## 実行タイミング
- **月1回**（毎月1日、product-creatorの後に実行）
- 手動実行も可: `node employees/line-builder/index.js`

## 生成するコンテンツ

### 1. LINE導線用X投稿テンプレート
```
投資の基礎から実践まで、毎週無料で配信してる。
Xだと文字数足りなくて伝えきれないことも多いから、
深い内容はLINEで出してる。

興味ある人はプロフに貼ってあるリンクから👇
```

### 2. LINE配信シナリオ（ステップ配信）
```json
{
  "step": 1,
  "day": 0,
  "title": "友だち追加ありがとう",
  "content": "kaizokuokabuです。Xではなかなか書けない...",
  "cta": "まずは自己紹介記事を読んでみて"
}
```

### 3. 週間配信プラン
- 月曜: 今週の注目イベント
- 水曜: 銘柄分析 or セクター分析
- 金曜: 週間振り返り＋来週の戦略

## 出力
- `employees/logs/line-scenarios/YYYY-MM.json`（月次シナリオ）
- LINE導線用投稿テンプレート → content-posterが参照

## 導線設計の原則
- X投稿で価値を示す → 「もっと深い話はLINEで」
- 強引な誘導はしない（ブランド毀損防止）
- 無料コンテンツで信頼構築 → 有料Noteへの自然な導線

## 制約
- 「無料LINE登録で〇〇プレゼント」系の煽り文句は禁止
- 情報商材的な表現は絶対に使わない
- config/brand-voice.md の「使わない表現」に準拠

## 注意
- LINE公式アカウントの開設はユーザー（社長）が手動で行う
- このモジュールはシナリオ生成のみ（LINE APIとの連携は将来対応）
