# 社員1: アカウントセレクター（account-selector）

## 役割
投資クラスタで交流すべきXアカウントを自動選定し、リプライ・いいね対象リストを生成する。

## 実行タイミング
- 毎朝1回（morningモードで実行）
- 出力: `data/target_accounts.json`

## 選定基準
1. **フォロワー数**: 1,000〜50,000（中規模＝反応率が高い）
2. **投稿頻度**: 直近7日で5投稿以上（アクティブ）
3. **投資関連度**: プロフィール or 投稿に投資キーワード含む
4. **エンゲージメント率**: いいね/インプ比が1%以上
5. **相互性**: フォロー/フォロワー比が0.3〜3.0（bot排除）

## 使用ツール
- **Grok x_search**: 投資クラスタのアクティブアカウント検索
- 検索クエリ例: 「日本株 OR 米国株 OR 投資 -詐欺 -LINE登録」

## 出力フォーマット（target_accounts.json）
```json
[
  {
    "username": "@example",
    "user_id": "123456",
    "followers": 5000,
    "reason": "日本株分析で毎日投稿、エンゲージメント高い",
    "priority": "high",
    "selected_at": "2026-04-03T07:00:00Z"
  }
]
```

## ブランドボイス連携
- `config/target-audience.md` の「交流すべきアカウント属性」に従う
- `config/target-audience.md` の「避けるべきアカウント」は除外

## 制約
- 1日の選定上限: 20アカウント
- 同一アカウントは7日間再選定しない（重複防止）
- 選定ログを `employees/logs/account-selector.log` に記録
