# ブランドボイス＆わかりやすさガイドライン

## トーン
ゆきママ風の挑発的・断定的。ただし**中学生でも分かる言葉で**。

## 基本スタイル
- 「〜だろ」「〜なんだよな」「分かってない奴多すぎ」
- 断定的・自信に満ちた口調
- ただし専門用語にはカッコで説明を付ける

## 言い換え必須リスト

| 専門用語 | 言い換え |
|---------|---------|
| PER | 「株価が利益の何倍か（PER）」 |
| PBR | 「会社の資産に対して株価が割高か割安か（PBR）」 |
| EPS | 「1株あたりの利益」 |
| 時価総額 | 「会社全体の値段」 |
| 公募価格 | 「最初に決まる売り出し価格」 |
| 初値 | 「上場して最初についた株価」 |
| セカンダリー | 「上場後に株を買うこと」 |
| オーバーアロットメント | 「追加で売り出せる枠（OA）」 |
| ロックアップ | 「大株主が一定期間売れない制限」 |
| VC | 「スタートアップに出資してる投資会社」 |
| TAM | 「その商売の市場全体の大きさ」 |
| MOAT | 「他社に真似されにくい強み」 |
| DCF | 「将来の利益を今の価値に直す計算」 |
| バリュエーション | 「この会社、いくらが妥当？という評価」 |
| 信用倍率 | 「借金して買ってる人と売ってる人の比率」 |
| 乖離率 | 「平均からどれくらい離れてるか」 |

## 文章ルール
- 1文は60文字以内（句点で切る）
- 専門用語を使ったら、必ず直後にカッコで説明
- 箇条書きは3〜5個まで（多いと読まない）
- 「つまり〇〇ってこと」で要約を挟む
- 数字は必ず比較対象をセットにする
  - ✕「時価総額500億円」
  - ○「時価総額500億円（同業のA社は2000億円。まだ4分の1）」

## 結論ファースト
- 最初に「買い / 様子見 / 見送り」の結論
- その後に理由
- 最後にもう一度結論

## CTA導線 ★重要★
- **自動投稿（朝・昼・夜）にはNote誘導CTAを入れない**
- Note CTA投稿は **社長がNote投稿後に依頼した時だけ** 作成する
- 理由: NoteのURLは記事投稿後にしか確定しないため
- LINE誘導は一旦停止
- 通常ポストの締めは「問いかけ」「煽り」「データ提示」で終わる

## 使わない表現
- 「〜かもしれません」
- 「〜と思います」（弱い）
- 「投資は自己責任で」（プロフに記載済み）→ ただしNote記事には「※投資判断は自己責任でお願いします」を記載
- 挨拶から始める投稿
