# 検索最適化キーワード集

## キーワード埋め込みルール
以下のキーワードを投稿内に**自然に本文内へ**含める（ハッシュタグではなく本文内に）

## 必須キーワード（銘柄投稿時）
- 銘柄名（正式名称 + 略称 + 証券コード）
  例: 「Timee（タイミー/215A）」「KOKUSAI ELECTRIC（6525）」
- 「IPO」「新規上場」「初値」「初値予想」「セカンダリー」「公募価格」
- 「上場予定」「上場日」「〇月上場」
- 旬のワード: 「テンバガー」「成長株」「決算」

## ハッシュタグ運用
- 1投稿に最大2個まで（多いと逆効果）
- 常用: #IPO #新規上場
- 状況に応じて: #初値予想 #IPO分析 #〇〇（銘柄名）
- 絶対使わない: #投資初心者 #副業（ターゲット外）

## トレンド活用
- 毎朝Xのトレンド確認、投資・経済関連トレンドワードがあれば当日ポストに絡める
- 決算発表日 → IPO時からの振り返りポスト
- 大型IPO前 → カウントダウン投稿（3日前・前日・当日）

## 投稿タイミング最適化
- 上場予定銘柄の分析 → 上場2週間前〜前日（検索需要ピーク）
- 初値結果速報 → 上場当日の前場終了直後
- セカンダリー分析 → 上場後1週間・1ヶ月・3ヶ月の節目
