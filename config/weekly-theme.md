# 週間テーマカレンダー

## 運用ルール
- 毎週月曜に当週テーマを更新
- テーマはあくまで「軸」。ニュース速報があればそちらを優先
- 1日の投稿の50%以上をテーマに沿わせる

## 2026年4月 第1週（4/1〜4/7）

| 曜日 | メインテーマ | サブテーマ |
|------|-------------|-----------|
| 月 | 新年度スタート・4月相場の傾向 | セルインメイ前の戦略 |
| 火 | 決算プレビュー（3月決算企業） | 注目セクター |
| 水 | 米国市場動向・ドル円 | テック株 vs バリュー株 |
| 木 | 日本株個別銘柄分析 | 高配当・優待 |
| 金 | 週間マーケット振り返り | 来週の注目イベント |
| 土 | 投資初心者向けコンテンツ | 投資マインドセット |
| 日 | 来週の戦略・ポートフォリオ | 長期投資の考え方 |

## テーマローテーション（月次）
1. マクロ経済（金利・為替・GDP）
2. セクター分析（半導体・AI・エネルギー）
3. 個別銘柄深掘り（決算・チャート）
4. 投資手法・戦略論
5. 海外市場（米国・中国・インド）
6. IPO・新規上場
7. 配当・優待・NISA

- 5月: 決算ラッシュ

- 8月: 夏枯れ相場
- 9月: 中間配当権利確定
- 11月: 米国感謝祭・ブラックフライデー
